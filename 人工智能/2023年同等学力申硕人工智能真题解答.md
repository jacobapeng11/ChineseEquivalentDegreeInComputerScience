# 2023年同等学力申硕人工智能真题解答

> 本文件只保留 2023 年试卷中的人工智能部分，并整理为便于复习的详细解析版。
> 说明：证明题中的符号由原始 PDF 转写而来，个别位置存在 OCR 噪声。下面按题目可确认的推理结构给出可得分的标准写法与答题步骤。

## 一、单项选择题（每题 2 分，共 8 分）

### 1. 使用神经网络求解分类问题，常用的训练手段是什么？

**答案：B 调整神经网络的连接权值**

**解析：**

- 神经网络学习的本质是根据训练样本不断修正各层之间的权值和偏置。
- 增减层数会改变模型容量，但不是训练分类器的核心操作。
- 输入节点个数通常由特征维度决定，也不是“求得分类问题的解”的主要手段。

### 2. 本体（Ontology）是一种什么表示方式？

**答案：C 概念和概念间关系的表达措施**

**解析：**

- 本体强调对领域内概念、属性、层次结构及相互关系的形式化描述。
- 它的目标是支持知识共享、语义一致和机器可理解，不是单独的一种推理策略。

### 3. 从理论上讲，是否不存在当今计算机不能求解的问题？

**答案：B 不是**

**解析：**

- 计算理论表明存在不可计算问题，例如停机问题。
- 因此不能说“所有问题都能由计算机求解”。
- 这类题目考查的是可计算性理论的基本结论。

### 4. 类似百度、Google 一类知识问答系统早期大量采用的重要技术是什么？

**答案：C 简朴的模式匹配措施**

**解析：**

- 这道题考查经典问答系统的工程实现思路。
- 在早期或规则驱动的知识问答场景中，大量系统依赖检索、模板匹配和模式匹配来定位答案。
- 现代系统会更多结合语义理解与知识推理，但原题对应的标准答案是模式匹配。

## 二、证明题（7 分）

题目要求：试用线性归结法，由给定前提证明目标结论成立。

### 得分点拆解

- 正确把前提与结论否定化为子句集，占 3 分。
- 给出一条合法的线性归结链，并最终推出空子句，占 4 分。

### 标准作答思路

1. 先把前提 $A_1, A_2, A_3$ 全部化为前束范式，再消去蕴含符号。
2. 对量词标准化后做 Skolem 化，最后改写成合取范式，拆成若干子句。
3. 将待证结论 $B$ 取否定，加入子句集。
4. 采用线性归结法时，后继归结步骤始终以上一步得到的子句为中心继续归结。
5. 若最终推出空子句 $\square$，则说明原结论成立。

### 考场可直接书写的模板

**第 1 步：写子句集**

- 将 $A_1, A_2, A_3$ 与 $\neg B$ 化成子句集 $S$。
- 只要子句化过程正确、谓词与变量对应合理，这一步即可拿到大部分证明题基础分。

**第 2 步：写归结链**

- 先从含有目标否定文字的子句开始。
- 再与前提中能够消去该文字的子句做归结。
- 每一步都写清楚“由哪两个子句归结得到什么新子句”。

**第 3 步：写结论**

- 最后一行写“得空子句 $\square$，故原命题得证”。

### 复习提示

- 这类题的关键不在于唯一的归结顺序，而在于过程合法、线性、最终归到空子句。
- 如果原题公式抄写较乱，卷面上优先保证：子句化正确、步骤完整、结论明确。

## 三、问答题（每题 5 分，共 15 分）

### 1. 近年来人工智能学科的研究热点，主要在哪几种方面？

**参考答案：**

可从下列方向作答，答出 3 点并展开即可拿到高分：

- 机器学习与深度学习：研究如何从数据中自动学习分类、预测和表征模型。
- 数据挖掘与知识发现：从海量数据中发现模式、规则和潜在知识。
- Agent 与多智能体系统：研究自治体在复杂环境中的感知、决策、协作与竞争。
- 自然语言处理：研究机器对文本、语音、语义和上下文的理解与生成。
- 计算机视觉与模式识别：研究图像、视频等信息的识别、检测和理解。

**答题建议：**

- 如果时间紧，可采用“方向 + 一句话解释”的结构。
- 这道题本质是送分题，答案不要求唯一，但要写成“成体系的几个方向”，不要只堆名词。

### 2. 阐明 Agent 概念，以及多 Agent 系统的重要研究内容。

**参考答案：**

**Agent 的概念：**

- Agent 是处于某种环境中的计算实体。
- 它能够感知环境，并根据自身目标自主地采取行动。
- Agent 一般具有自主性、反应性、主动性和社会性等特点。

**多 Agent 系统的重要研究内容：**

- Agent 个体模型：研究单个 Agent 的知识表示、感知、决策和行为机制。
- Agent 间通信与协作：研究多个 Agent 如何交换信息、协商和分配任务。
- 群体求解机制：研究多个 Agent 如何通过协同完成单个 Agent 难以独立完成的问题。
- 冲突消解与协调控制：研究资源竞争、目标冲突和全局一致性问题。

**可直接背诵的精简表述：**

Agent 是能够在环境中自主感知并行动的智能实体；多 Agent 系统主要研究多个 Agent 的建模、通信、协作、协调以及群体求解机制。

### 3. 什么叫一种归结方法是完备的？举出一种不完备的归结策略。

**参考答案：**

**完备性的含义：**

- 若一个结论逻辑上可由前提推出，那么使用该归结方法最终一定能够证明出来，这种性质就称为完备性。
- 简单说，就是“只要命题真的可证，这种方法就不会漏证”。

**不完备的归结策略示例：**

- 单元归结策略是不完备的。

**解析：**

- 归结原理本身在一阶逻辑中是重要的自动推理方法。
- 但为了提高效率，实际会采用不同的归结策略。
- 有些策略虽然更快，却可能漏掉本来可以证明的结论，因此称为不完备。

## 四、总评

2023 年人工智能部分的命题特点比较鲜明：

- 选择题偏基础概念。
- 证明题考线性归结法。
- 问答题偏宏观综述与概念辨析。

复习时建议优先掌握以下三类模板：

- 知识表示与推理基础概念辨析。
- 线性归结法的子句化与归结步骤模板。
- Agent、研究热点、完备性这类 5 分简答题模板。
