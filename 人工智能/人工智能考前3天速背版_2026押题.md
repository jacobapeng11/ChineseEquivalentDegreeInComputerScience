# 人工智能考前 3 天速背版：2026 押题版

> 用法：这不是教材，是最后 3 天的取分清单。
>
> 目标：只背 2026 最可能直接兑现的模板和结论。

## 一、先记结论：2026 最可能怎么考

### 1. 最可能结构

- 选择题：3 题，6 分
- 证明题：1 题，10 分
- 计算题：1 题，8 分
- 简答题：2 题，12 分

### 2. 最值得押的六块

1. 谓词逻辑规则链证明
2. CER / CF 可信度计算
3. 知识表示辨析
4. 搜索算法比较
5. CNN 层次与作用
6. Agent / 完备性 / NLP 的轮换回潮

### 3. 最重要的策略判断

- 不要赌“2025 考过了，2026 就不考”。
- 真正稳定的是 2024/2025 的题型结构，而不是某个具体题干。
- 2026 最值得押的增量信号，是把 2023 考过但 2025 没考的 Agent、完备性、研究热点或 NLP 拉回一题。

## 二、考前 3 天安排

### 第 1 天：证明题 + 知识表示

今天只做三件事：

1. 把自然语言翻成谓词公式的常见写法背熟。
2. 把“已知事实 → 中间结论 → 最终结论”的证明模板刷熟。
3. 把本体、语义网络、框架、产生式规则、谓词逻辑的区别刷一遍。

今天必须会写：

- 所有 A 都是 B
- 存在一个 A 是 B
- 不是每个 A 都是 B
- 至多一个、恰好一个
- “由规则 R1 和已知事实 F1，可得中间结论 P(a)” 这种链式句子

### 第 2 天：CER / CF + 搜索

今天只做三件事：

1. 把 AND 取 min、OR 取 max 练到下意识反应。
2. 把 CER(E) 乘规则可信度得到结论 CF 的步骤练熟。
3. 把 OPEN / CLOSED、DFS / BFS、A* 的比较表过一遍。

今天必须会记：

- AND：取最小值
- OR：取最大值
- 结论可信度：前提综合可信度 × 规则可信度
- DFS：深、栈、空间小、不一定最优
- BFS：广、队列、空间大、单位代价下最优

### 第 3 天：CNN + 回潮理论题

今天只做三件事：

1. 背熟 CNN 的主要层及作用。
2. 过一遍 Agent / 多 Agent、归结完备性、NLP 流程。
3. 限时做 1 套押题卷。

今天必须会答：

- 卷积层做什么
- 激活层做什么
- 池化层做什么
- 全连接层做什么
- Agent 的定义和多 Agent 系统研究内容
- 完备性的含义与不完备策略示例
- NLP 的词法、语法、语义分析

## 三、最该背的 12 个结论

1. 决策树不是深度学习架构。
2. 贝叶斯网络属于不确定推理方法。
3. 谓词逻辑不是结构化知识表示方法。
4. 本体用于表示概念及概念间关系。
5. OPEN 表是待扩展结点表。
6. CLOSED 表是已扩展结点表。
7. DFS 常按栈处理，BFS 常按队列处理。
8. CNN 常考卷积层、激活层、池化层、全连接层。
9. Agent 是能在环境中感知并自主行动的智能实体。
10. 完备性就是“只要可证，就不会漏证”。
11. CER / CF 题里 AND 取 min，OR 取 max。
12. 2026 最可能继续保留证明题和可信度计算题。

## 四、考场直接套的四个模板

### 1. 谓词逻辑证明题

先写：

- 已知事实
- 规则 1 推出什么
- 规则 2 推出什么
- 最终由两条中间结论推出目标结论

### 2. CER / CF 计算题

先算括号内 OR，再算整体 AND，最后乘规则可信度。

### 3. 搜索简答题

先解释 OPEN / CLOSED，再比较 DFS 与 BFS 的扩展顺序、空间和最优性。

### 4. CNN / Agent / NLP 理论题

按“定义 + 组成 / 特点 + 作用 / 研究内容”三段式写。
