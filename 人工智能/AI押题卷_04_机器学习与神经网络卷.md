## 三、计算和简答题（共20分）

### 1. 计算题（8分）

对于一个简单的神经网络：

- 输入层 2 个神经元
- 隐藏层 2 个神经元
- 输出层 1 个神经元
- 输入 x = [1, 1]
- 输入到隐藏的权重矩阵 W1 = [[0.1, 0.2], [0.3, 0.4]]
- 偏置 b1 = [0.1, 0.1]
- 隐藏到输出的权重 W2 = [0.5, 0.6]
- 偏置 b2 = 0.2

使用 Sigmoid 激活函数，计算最终输出。

### 2. 简答题一（6分）

解释支持向量和核函数的含义，以及它们在 SVM 中的作用。

### 3. 简答题二（6分）

比较 RNN、LSTM 和 Transformer 在处理序列数据时的主要特点。

### 二、证明题参考答案

由 3 可知：Bird(Tweety)。

由规则1：Bird(x) → Animal(x)，得：Animal(Tweety)。

由规则2：Animal(x) → NeedFood(x)，得：NeedFood(Tweety)。

结论成立。

### 三、计算和简答题答案

#### 1. 计算题

按输入行为向量、W1 每列对应一个隐藏神经元计算：

xW1 = [1, 1]

得到：

- h1_in = 1×0.1 + 1×0.3 + 0.1 = 0.5
- h2_in = 1×0.2 + 1×0.4 + 0.1 = 0.7

Sigmoid 后：

- h1 = 1 / (1 + e^(-0.5)) ≈ 0.6225
- h2 = 1 / (1 + e^(-0.7)) ≈ 0.6682

输出层输入：

o_in = 0.6225×0.5 + 0.6682×0.6 + 0.2

= 0.31125 + 0.40092 + 0.2

= 0.91217

最终输出：

o = 1 / (1 + e^(-0.91217)) ≈ 0.7134

答案：最终输出约为 0.713

#### 2. 简答题一

- 支持向量：距离最优分类超平面最近、对分类面位置起决定作用的样本点
- 核函数：把原空间样本映射到高维特征空间并计算内积的函数

作用：

1. 支持向量决定最大间隔超平面
2. 核函数使 SVM 能处理非线性可分问题

#### 3. 简答题二

- RNN：按时间顺序递归处理序列，能利用历史信息，但长依赖能力弱
- LSTM：在 RNN 基础上加入门控机制，缓解梯度消失，适合长依赖
- Transformer：基于自注意力，可并行处理序列，长距离依赖建模能力更强

## 押题原因

1. 神经网络基础、SVM、深度学习架构都在考纲中明确出现。
2. 近年选择题多次围绕“深度学习架构”“神经网络作用”“结构化表示”出题。
3. 如果命题人今年想拉开区分度，最容易加一题小型前向传播计算。

## 最佳解题方法

1. 神经网络计算题先明确矩阵含义，再逐层算，不要混淆“按行”还是“按列”。
2. 支持向量题必须写出“最近样本点”四个字，核函数题必须写“高维映射/非线性”。
3. RNN/LSTM/Transformer 比较题按“机制 + 优点 + 局限”三点作答。
