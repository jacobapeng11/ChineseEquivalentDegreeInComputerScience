## 二、证明题（10分）

已知：

1. 通过人工智能考试并获奖的人可以获得奖学金：Scholar(x) ← Pass(x, AI) ∧ Win(x)
2. 努奋学习或幸运的人可以通过人工智能考试：Pass(x, AI) ← Study(x) ∨ Lucky(x)

## 参考答案与评分点

### 一、选择题答案

1. C
2. A
3. C

### 二、证明题参考答案

设：

- Scholar(x)：x 获得奖学金
- Pass(x, AI)：x 通过人工智能考试
- Win(x)：x 获奖
- Study(x)：x 努奋学习
- Lucky(x)：x 幸运

已知 Lucky(WangMin)。

由规则4：Lucky(x) → Win(x)，得：Win(WangMin)。

又因为 Lucky(WangMin) 成立，所以 Study(WangMin) ∨ Lucky(WangMin) 成立。

由规则2：Pass(x, AI) ← Study(x) ∨ Lucky(x)，得：Pass(WangMin, AI)。

再由规则1：Scholar(x) ← Pass(x, AI) ∧ Win(x)，结合

- Pass(WangMin, AI)
- Win(WangMin)

得：Scholar(WangMin)。

### 三、计算和简答题答案

#### 1. 计算题

TF = 5 / 100 = 0.05

IDF = log(1000 / 50) = log(20) ≈ 1.30

TF-IDF = 0.05 × 1.30 = 0.065

答案：TF-IDF ≈ 0.065

#### 2. 简答题一

自然语言理解的四个主要层次：

1. 词法/形态学分析：分词、词形变化、词性等
2. 语法/句法分析：句子结构与语法关系
3. 语义分析：理解词义和句子含义
4. 语用分析：结合上下文理解说话意图

#### 3. 简答题二

词向量表示的优势：

1. 能表示语义相似性
2. 维度低于 One-Hot
3. 有利于模型学习分布式特征
4. 可支持词间关系运算
5. 泛化能力优于简单词频统计

## 押题原因

1. 考纲明确要求掌握自然语言理解过程、基本词切分和语法分析。
2. 2023 选择题已经出现过自然语言语义理解方向，NLP 在人工智能卷中不是边缘知识点。
3. TF-IDF、语言模型、词向量是最容易命成选择/简答/基础计算的 NLP 小题。

## 最佳解题方法

1. TF-IDF 一律按三步：先算 TF，再算 IDF，最后相乘。
2. log(10)=1，log(20)≈1.30，log(100)=2 这几个值最好直接记住。
3. NLP 四层次题不要展开太多，每层写“名称 + 核心任务”即可。
4. 词向量优势题一定要提到“语义相似性”和“低维稠密表示”。
