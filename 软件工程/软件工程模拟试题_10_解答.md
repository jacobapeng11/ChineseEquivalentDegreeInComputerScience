# 软件工程模拟试题 10 - 详细解答

## 一、选择题解答（每题2分，共20分）

1. **答案：D**
   **解析：** 软件复杂性体现在多个方面，包括问题域的复杂性、逻辑复杂性、数据复杂性、管理复杂性等，其中问题域和管理复杂性是主要体现。

2. **答案：D**
   **解析：** 软件配置管理的基本活动包括配置标识、配置控制、配置状态报告和配置审计，不包括配置优化。

3. **答案：B**
   **解析：** 合成复用原则（Composite Reuse Principle）强调优先使用合成或聚合关系而非继承关系来复用代码。

4. **答案：A**
   **解析：** 在V模型中，详细设计对应单元测试，概要设计对应集成测试。

5. **答案：C**
   **解析：** CMMI已管理级（第4级）的特征是过程是定量管理的，使用统计和定量技术对过程进行控制。

6. **答案：C**
   **解析：** 判定覆盖（分支覆盖）是白盒测试技术，而等价类划分、边界值分析、因果图都是黑盒测试技术。

7. **答案：B**
   **解析：** 关键路径是项目网络图中的最长路径，决定了项目的最短完成时间。

8. **答案：D**
   **解析：** 用户故事的3个C是Card（卡片）、Conversation（对话）、Confirmation（确认）。

9. **答案：B**
   **解析：** 基线的主要作用是为后续工作提供参考标准，作为变更控制的基准点。

10. **答案：D**
    **解析：** 性能效率包括时间特性、资源利用率、容量等子特性，易用性是独立的质量特性。

## 二、填空题解答（每空2分，共20分）

11. **答案：** 软件质量保证
    **解析：** 软件质量保证是为改善软件质量而进行的活动。

12. **答案：** 过程
    **解析：** 软件工程的三大要素是方法、工具和过程。

13. **答案：** 产品待办列表（Product Backlog）
    **解析：** 在Scrum中，产品待办列表是按优先级排序的需求列表。

14. **答案：** 序列（Sequence）或通信（Communication）
    **解析：** 序列图或通信图用于描述系统动态行为和对象交互。

15. **答案：** 输入
    **解析：** 黑盒测试关注输入和输出的关系，不关心内部逻辑。

16. **答案：** 适应性
    **解析：** 为适应环境变化而进行的修改属于适应性维护。

17. **答案：** 圈复杂度（或McCabe复杂度）
    **解析：** 圈复杂度用于衡量代码的复杂程度。

18. **答案：** EV（挣值）
    **解析：** 挣值管理的三个参数是PV（计划值）、AC（实际成本）、EV（挣值）。

19. **答案：** 软件工程化
    **解析：** 软件工程化是将软件工程方法系统地应用到软件开发过程。

20. **答案：** 变更控制
    **解析：** 变更控制是建立和控制对软件项更改的活动。

## 四、综合分析题解答（每题15分，共30分）

24. **答案：**
    (1) 项目组织结构（4分）：
    - 采用矩阵式组织结构（1分）
    - 项目经理负责整体管理（1分）
    - 技术负责人负责技术决策（1分）
    - 各子系统设立子项目经理（1分）
    
    (2) 开发流程（4分）：
    - 采用敏捷开发方法，如Scrum（2分）
    - 将项目分解为多个Sprint（1分）
    - 每个Sprint包含需求、设计、开发、测试活动（1分）
    
    (3) 质量保证（4分）：
    - 建立代码审查机制（1分）
    - 实施自动化测试（1分）
    - 制定编码规范（1分）
    - 进行持续集成（1分）
    
    (4) 风险管理（3分）：
    - 识别技术风险、进度风险、人员风险（1分）
    - 制定风险应对计划（1分）
    - 定期进行风险评估（1分）

25. **答案：**
    (1) 云原生特点和关键技术（4分）：
    - 特点：容器化、微服务、弹性伸缩、DevOps（2分）
    - 关键技术：容器、微服务架构、API优先、不可变基础设施（2分）
    
    (2) 容器化影响（4分）：
    - 开发：环境一致性、快速部署（2分）
    - 部署：简化部署流程、提高资源利用率（2分）
    
    (3) 微服务与单体架构比较（4分）：
    - 微服务优势：独立开发部署、技术多样性、可扩展性（2分）
    - 微服务劣势：分布式复杂性、运维难度大（1分）
    - 单体优势：简单开发运维（1分）
    
    (4) 技能要求变化（3分）：
    - 需要掌握容器技术（1分）
    - 需要了解分布式系统（1分）
    - 需要DevOps技能（1分）
