9. 齐次方程aₙ = 3a_{n-1} - 2a_{n-2}，特征方程x² - 3x + 2 = 0，(x-1)(x-2) = 0，x₁=1, x₂=2
   齐次通解：aₙ^h = α·1ⁿ + β·2ⁿ = α + β·2ⁿ
   
   非齐次特解：由于2是特征根，设aₙ^p = γn·2ⁿ
   代入原方程：γn·2ⁿ = 3γ(n-1)·2^{n-1} - 2γ(n-2)·2^{n-2} + 2ⁿ
   两边同除以2^{n-2}：4γn = 6γ(n-1) - 2γ(n-2) + 4
   4γn = 6γn - 6γ - 2γn + 4γ + 4 = 4γn - 2γ + 4
   所以-2γ + 4 = 0，γ = 2
   特解：aₙ^p = 2n·2ⁿ = n·2^{n+1}
   
   通解：aₙ = α + β·2ⁿ + n·2^{n+1}
   
   由a₀ = 1: α + β = 1
   由a₁ = 2: α + 2β + 4 = 2，即α + 2β = -2
   解得：β = -3，α = 4
   所以aₙ = 4 - 3·2ⁿ + n·2^{n+1} = 4 + 2ⁿ(2n - 3)

12. 解：
    设U = {1, 2, ..., 1000}
    A = U中被2整除的数集合，|A| = ⌊1000/2⌋ = 500
    B = U中被3整除的数集合，|B| = ⌊1000/3⌋ = 333
    C = U中被5整除的数集合，|C| = ⌊1000/5⌋ = 200
    
    |A∩B| = ⌊1000/lcm(2,3)⌋ = ⌊1000/6⌋ = 166
    |A∩C| = ⌊1000/lcm(2,5)⌋ = ⌊1000/10⌋ = 100
    |B∩C| = ⌊1000/lcm(3,5)⌋ = ⌊1000/15⌋ = 66
    |A∩B∩C| = ⌊1000/lcm(2,3,5)⌋ = ⌊1000/30⌋ = 33
    
    由容斥原理：
    |A∪B∪C| = |A| + |B| + |C| - |A∩B| - |A∩C| - |B∩C| + |A∩B∩C|
    = 500 + 333 + 200 - 166 - 100 - 66 + 33 = 734
    
    所以不能被2、3或5整除的整数个数 = |U| - |A∪B∪C| = 1000 - 734 = 266

### 四、证明题
13. 证明：
    在简单无向图中，任意两个不同顶点之间最多有一条边，且没有自环。
    n个顶点中任选2个的组合数为C(n,2) = n(n-1)/2。
    每个这样的顶点对最多贡献1条边。
    
    所以边数m ≤ n(n-1)/2。
    
    等号成立当且仅当每对不同顶点之间都恰好有一条边，这正是完全图Kₙ的定义。
    所以m = n(n-1)/2当且仅当G是完全图。

14. 证明：
    设a是Zₙ中的元素，a的阶为d意味着最小的正整数k使得ka ≡ 0 (mod n)是d。
    这等价于d|n且a = n/d · j，其中gcd(j, d) = 1。
    
    令b = a·(n/d)，则a = b·(n/d)。如果ord(a) = d，则ord(b·(n/d)) = d。 
    由于ord(b·(n/d)) = n/gcd(b·(n/d), n) = n/(n/gcd(b,n/d)) = gcd(b,n/d)·(n/d)，
    这个等于d当且仅当gcd(b,n/d) = d/(n/d)·(n/d) = d·(n/d)/n·n = d·(n/d)·n/n = d·(n/d)，这不对。
    
    更直接地：在Zₙ中，元素a的阶是n/gcd(a,n)。
    令ord(a) = d，即n/gcd(a,n) = d，所以gcd(a,n) = n/d。
    令k = gcd(a,n) = n/d，那么a = k·j，其中gcd(j, n/k) = gcd(j, d) = 1。
    
    所以阶为d的元素a = (n/d)·j，其中j ∈ {1, 2, ..., d}且gcd(j, d) = 1。
    
    这样的j的个数是φ(d)。
    
    因此，Zₙ中阶为d的元素个数为φ(d)。
