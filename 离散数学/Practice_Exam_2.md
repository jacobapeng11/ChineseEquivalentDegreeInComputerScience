## 参考答案

### 一、选择题
1. B - 设A的真值表中有m个成真赋值，则主析取范式含m个极小项；有2ⁿ-m个成假赋值，故主合取范式含2ⁿ-m个极大项
2. A - 等价类要么相等，要么不交；aRb⟺[a]=[b]⟺[a]∩[b]≠∅
3. C - 有向简单图中，任意两个顶点之间最多有两条方向相反的有向边，所以最多n(n-1)条边
4. D - 由Lagrange定理，元素的阶必须整除群的阶。6的因子有1,2,3,6，不含4
5. B - 正方形4边着色在C₄作用下：
   - e: 2⁴=16个不动点
   - 旋转90°,270°: 2¹=2个不动点（4条边必须同色）
   - 旋转180°: 2²=4个不动点（对边必须同色）
   - 本质不同着色数 = (1/4)(16+2+4+2) = 24/4 = 6

### 三、计算题
11. 解：
    按边权重递增排序：(B,C)=1, (B,E)=3, (B,D)=5, (C,D)=5, (A,B)=6, (A,E)=7, (C,E)=8, (D,E)=9
    
    按Kruskal算法依次选择边：
    1. 选择(B,C)=1（不会形成回路）
    2. 选择(B,E)=3
    3. 选择(B,D)=5
    4. 考虑(C,D)=5，B-C-D已连通，选(C,D)会形成回路，跳过
    5. 选择(A,B)=6
    此时已有4条边，n=5个顶点的生成树需要4条边，算法结束
    
    最小生成树的边集：{(B,C), (B,E), (B,D), (A,B)}，权重和为1+3+5+6=15

12. 解：
    设G(x) = Σ n²xⁿ (n从0到∞)为序列{n²}的普通生成函数
    已知序列{1, 1, 1, ...}的生成函数为1/(1-x)
    序列{0, 1, 2, 3, ...}的生成函数为x/(1-x)²（对1/(1-x)求导再乘x）
    序列{0, 1, 4, 9, ...}的生成函数为x(1+x)/(1-x)³（对x/(1-x)²求导再乘x）
    
    或者：
    从序列{1, 1, 1, ...}的生成函数1/(1-x)开始
    序列{0, 1, 2, 3, ...}的生成函数为x·d/dx[1/(1-x)] = x·1/(1-x)² = x/(1-x)²
    序列{0, 0, 1, 4, 9, ...}的生成函数为x·d/dx[x/(1-x)²] = x·((1-x)²+2x(1-x))/(1-x)⁴
    = x·(1-x+2x)/(1-x)³ = x(1+x)/(1-x)³
    
    所以G(x) = x(1+x)/(1-x)³

### 四、证明题
13. 证明：
    用反证法。假设图G不连通，则G至少包含两个连通分量。
    设其中一个连通分量有k个顶点（1≤k≤n-1），其余连通分量有n-k个顶点。
    
    任取该连通分量中一个顶点v，则v最多只能与该连通分量中其他k-1个顶点相邻，
    所以deg(v) ≤ k-1。
    
    由题设条件，deg(v) ≥ (n-1)/2，所以 k-1 ≥ (n-1)/2，即 k ≥ (n-1)/2 + 1 = (n+1)/2。
    
    同样，任取其余连通分量中一个顶点u，则u最多只能与该连通分量中其他n-k-1个顶点相邻，
    所以deg(u) ≤ n-k-1。
    
    由题设条件，deg(u) ≥ (n-1)/2，所以 n-k-1 ≥ (n-1)/2，即 k ≤ n-1-(n-1)/2 = (n-1)/2。
    
    但由 k ≥ (n+1)/2 和 k ≤ (n-1)/2，得到 (n+1)/2 ≤ (n-1)/2，即 n+1 ≤ n-1，即 1 ≤ -1，矛盾。
    
    所以原假设错误，图G是连通的。

14. 证明：
    S(n,k)表示将n个相异元素分为k个非空不相交集合的方法数。
    
    考虑第n个元素的归属：
    1. 第n个元素单独成为一个集合：这时前n-1个元素需要分为k-1个非空集合，方法数为S(n-1,k-1)
    2. 第n个元素加入已有的集合：先将前n-1个元素分为k个非空集合，方法数为S(n-1,k)，
       然后将第n个元素加入k个集合中的任意一个，方法数为k，总共k·S(n-1,k)种方法
    
    由于第n个元素要么单独成组，要么加入已有组，没有其他可能，且这两类方法互不重复，
    所以S(n,k) = S(n-1,k-1) + k·S(n-1,k)
