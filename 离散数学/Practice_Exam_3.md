### 四、证明题
13. 证明：
    设G是p阶群，其中p是素数，e是G的单位元。
    
    任取a∈G且a≠e，设a的阶为d，则d > 1（因为a≠e）。
    
    由Lagrange定理，a的阶d整除群G的阶p。由于p是素数，且d > 1，所以d = p。
    
    所以a的阶是p，这意味着元素a, a², a³, ..., aᵖ = e互不相同（否则若aⁱ = aʲ (0 < i < j ≤ p)，则a^{j-i} = e，而0 < j-i < p，这与a的阶是p矛盾）。
    
    因此，集合{a, a², a³, ..., aᵖ⁻¹, aᵖ=e}包含p个不同的元素，它们都在G中。
    但G只有p个元素，所以G = {a, a², a³, ..., aᵖ⁻¹, e}。
    
    这表明G是由a生成的循环群，即G = <a>。
    
    定义φ: Zₚ → G为φ([k]ₚ) = aᵏ，则φ是群同构：
    - φ是映射：每个[k]ₚ对应唯一aᵏ
    - φ是单射：若φ([k]ₚ) = φ([j]ₚ)，则aᵏ = aʲ，所以a^{k-j} = e，因为a的阶是p，所以p | (k-j)，即[k]ₚ = [j]ₚ
    - φ是满射：G中每个元素都是某个aᵏ
    - φ保持运算：φ([k]ₚ + [j]ₚ) = φ([k+j]ₚ) = a^{k+j} = aᵏaʲ = φ([k]ₚ)φ([j]ₚ)
    
    所以G ≅ Zₚ。

14. 证明：
    Catalan数Cₙ表示n+2边形的三角剖分数，也表示n对括号的匹配方案数。

    考虑n对括号的匹配方案。第一个括号'('必须与某个')'匹配，设与第2i个位置的')'匹配（其中i∈{1,2,...,n}）。

    这样，括号序列被分为：
    1. 从第2个到第2i-1个位置，共i-1对括号的匹配，方案数为C_{i-1}
    2. 从第2i+1个到第2n个位置，共n-i对括号的匹配，方案数为C_{n-i}

    对所有可能的i值（i从1到n）求和：
    Cₙ = Σ_{i=1}^n C_{i-1}·C_{n-i} = Σ_{i=0}^{n-1} Cᵢ·C_{n-1-i}

    这就是Catalan数的递推关系。

    初始条件C₀=1（空序列对应1种匹配）。
