## 一、谓词逻辑解答（3分）

**答案：**
∀x(P(x) → Q(x)) ∧ ¬∀x(Q(x) → R(x))

或者可以表示为：
∀x(P(x) → Q(x)) ∧ ∃x(Q(x) ∧ ¬R(x))

**解析：**
- "成绩达标的才有资格参加奥运会"：∀x(P(x) → Q(x))
- "不是每个参加奥运会的人都能获得奖牌"：¬∀x(Q(x) → R(x)) 或 ∃x(Q(x) ∧ ¬R(x))

### 填空题解答（10分）

偏序集相关解答：
- 最大链路长度：3（例如a≤b≤d或a≤b≤e）
- 最大链路个数：2（b→d, b→e或b→e, c→d等）
- 极大值：d,e（没有比它们更大的元素）
- 极小值：a（没有比它更小的元素）
- 最大值：不存在（d和e不可比较）
- 最小值：a（所有元素都比它大或等于它）
- 反链：长度最大的反链，例如{b,c}（2个互不可比较的元素）

组合数学：
- x₁+x₂+x₃+x₄+x₅+x₆ = 7的非负整数解个数：C(7+6-1, 6-1) = C(12,5) = 792
- x₁+x₂+x₃+x₄+x₅+x₆ = 7的正整数解个数：令yi = xi-1，转化为y₁+y₂+y₃+y₄+y₅+y₆ = 1，解个数为C(1+6-1, 6-1) = C(6,5) = 6

## 四、证明题解答（10分）

### 1. 对称差证明（7分）

**证明：** (A⊕B)×C = (A×C)⊕(B×C)

**证明：** 
设(x,y)∈(A⊕B)×C，则x∈(A⊕B)且y∈C
因此x∈(A-B)∪(B-A)且y∈C
即[x∈A且x∉B且y∈C] 或 [x∈B且x∉A且y∈C]

对于第一种情况：x∈A且x∉B且y∈C，则(x,y)∈A×C且(x,y)∉B×C，所以(x,y)∈(A×C)-(B×C)
对于第二种情况：x∈B且x∉A且y∈C，则(x,y)∈B×C且(x,y)∉A×C，所以(x,y)∈(B×C)-(A×C)

所以(x,y)∈[(A×C)-(B×C)]∪[(B×C)-(A×C)] = (A×C)⊕(B×C)

反过来，若(x,y)∈(A×C)⊕(B×C)，同样可证(x,y)∈(A⊕B)×C
故等式成立。

### 2. 图的证明（3分）

**证明：** 对于不含三角形的简单连通平面图，每个面的边界至少包含4条边。
设f为面数，根据握手定理，边数m与面数f的关系为：2m ≥ 4f（因为每个面边界的度数至少为4）
由欧拉公式：n - m + f = 2，得f = 2 + m - n
代入不等式得：2m ≥ 4(2 + m - n) = 8 + 4m - 4n
整理得：2m ≤ 4n - 8，即m ≤ 2n - 4
