### 四、证明题
13. 证明：
    设(B, ∨, ∧, ¬, 0, 1)是一个有限布尔代数，其中|B| = n。
    
    令A为B的所有原子组成的集合。在有限布尔代数中，每个非零元素可以唯一地表示为某些原子的并。
    
    定义映射φ: B → P(A)为：对任意x ∈ B，φ(x) = {a ∈ A | a ≤ x}（即x所包含的所有原子的集合）
    
    证明φ是同构：
    1) φ是映射：对每个x∈B，φ(x)是A的唯一子集
    2) φ是单射：若φ(x) = φ(y)，则x和y包含相同的原子，由原子表示的唯一性，x = y
    3) φ是满射：对任意S ∈ P(A)，令x = ∨_{a∈S} a，则φ(x) = S
    4) φ保持运算：
       φ(x ∨ y) = {a ∈ A | a ≤ x ∨ y} = {a ∈ A | a ≤ x 或 a ≤ y}（由于a是原子）
       = {a ∈ A | a ≤ x} ∪ {a ∈ A | a ≤ y} = φ(x) ∪ φ(y)
       类似可证φ(x ∧ y) = φ(x) ∩ φ(y)，φ(¬x) = A \ φ(x)
    
    所以B ≅ P(A)，即任意有限布尔代数都同构于某个幂集代数。

14. 证明：
    S(n,k)表示将n个相异元素划分为k个非空等价类的方法数。
    
    左边Σ_{k=0}^n S(n,k)表示将n个元素划分为任意个（1到n个）非空集合的方法总数。
    
    右边B(n)定义为n元集合的所有划分的个数。
    
    对于n元集合的任一划分，设它包含k个非空子集，则这种划分对应于S(n,k)中的一种情况，其中k可以是0到n中的任意值（但S(n,k)=0当k > n或k < 0）。
    
    所以n元集合的所有划分按照划分的子集个数分类计数，即：
    划分为1个子集的划分数 = S(n,1)
    划分为2个子集的划分数 = S(n,2)
    ...
    划分为n个子集的划分数 = S(n,n)
    划分为k个子集的划分数 = S(n,k) (当k > n时为0)
    
    因此，B(n) = S(n,1) + S(n,2) + ... + S(n,n) = Σ_{k=1}^n S(n,k) = Σ_{k=0}^n S(n,k)
    （因为S(n,0)=0当n > 0时，S(0,0)=1，符合B(0)=1）
    
    所以Σ_{k=0}^n S(n,k) = B(n)
