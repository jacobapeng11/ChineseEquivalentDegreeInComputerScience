12. 解：
    齐次方程：aₙ = 5a_{n-1} - 6a_{n-2}
    特征方程：x² - 5x + 6 = 0，(x-2)(x-3) = 0，x₁=2, x₂=3
    齐次通解：aₙ^h = α·2ⁿ + β·3ⁿ
    
    非齐次特解：由于2是特征根，设特解aₙ^p = γn·2ⁿ
    代入原方程：γn·2ⁿ = 5γ(n-1)·2^{n-1} - 6γ(n-2)·2^{n-2} + 2ⁿ
    两边同除以2^{n-2}：4γn = 10γ(n-1) - 6γ(n-2) + 4
    展开：4γn = 10γn - 10γ - 6γn + 12γ + 4 = 4γn + 2γ + 4
    所以2γ + 4 = 0，γ = -2
    特解：aₙ^p = -2n·2ⁿ
    
    通解：aₙ = α·2ⁿ + β·3ⁿ - 2n·2ⁿ
    
    初始条件：
    a₀ = α + β = 1
    a₁ = 2α + 3β - 4 = 1
    由第二个方程：2α + 3β = 5
    解得：α = -2, β = 3
    
    所以aₙ = -2·2ⁿ + 3·3ⁿ - 2n·2ⁿ = 3·3ⁿ - 2·2ⁿ(1+n)

### 四、证明题
13. 证明：
    将问题转换为6个顶点的完全图K₆的边着色问题（红边表示认识，蓝边表示不认识）。
    考虑任意一个顶点v，它关联其他5个顶点，于是有5条边从v出发。
    由鸽巢原理，这5条边中至少有⌈5/2⌉ = 3条边同色。不妨设为红色。
    设这3条红边是(v,a), (v,b), (v,c)。
    现考虑a, b, c之间的边：
    - 如果a, b, c之间有任何一条红边（如(a,b)），则v, a, b构成红色三角形（三个都相互认识）
    - 如果a, b, c之间的所有边都是蓝色，则a, b, c构成蓝色三角形（三个都相互不认识）
    
    因此，无论哪种情况，都存在一个同色三角形，即要么有3个人相互认识，要么有3个人相互不认识。

14. 证明：
    设S为n个元素的全排列集合，|S| = n!。
    设Aᵢ表示第i个元素在第i个位置上的排列组成的集合。
    那么错排数D(n)就是不在任何Aᵢ中的排列数，即|S - (A₁∪A₂∪...∪Aₙ)|。
    
    由容斥原理：
    D(n) = |S| - |A₁∪...∪Aₙ| 
         = |S| - Σ|Aᵢ| + Σ|Aᵢ∩Aⱼ| - Σ|Aᵢ∩Aⱼ∩Aₖ| + ... + (-1)ⁿ|A₁∩...∩Aₙ|
    
    其中：
    - |S| = n!
    - |Aᵢ| = (n-1)!（第i个位置固定，其余任意排列）
    - |Aᵢ∩Aⱼ| = (n-2)!（第i和第j个位置固定）
    - ...
    - |Aᵢ₁∩...∩Aᵢₖ| = (n-k)!
    
    因为有C(n,k)个k个集合的交，所以：
    D(n) = n! - C(n,1)(n-1)! + C(n,2)(n-2)! - ... + (-1)ⁿC(n,n)0!
         = n! - n!/1!(n-1)! + n!/2!(n-2)! - ... + (-1)ⁿn!/n!0!
         = n!(1/0! - 1/1! + 1/2! - ... + (-1)ⁿ/n!)
         = n!Σ_{k=0}^n (-1)ᵏ/k!
