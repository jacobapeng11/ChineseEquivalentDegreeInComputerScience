### 三、计算题
11. 解：
    A = {a, b, c}上的等价关系与A的划分一一对应。
    {a, b, c}的划分有：
    1) {{a}, {b}, {c}}
    2) {{a, b}, {c}}
    3) {{a, c}, {b}}
    4) {{b, c}, {a}}
    5) {{a, b, c}}
    
    所以有5个等价关系。
    
    以等价关系R₂ = {(a,a), (b,b), (c,c), (a,b), (b,a)}为例：
    R₂的矩阵为：
    [1 1 0]
    [1 1 0]
    [0 0 1]

12. 解：
    Dijkstra算法：
    初始化：A到A距离0，其他顶点距离∞
    S = {}
    
    步骤1：选择A（距离0），更新邻接点：B(2), C(4)
    S = {A}
    
    步骤2：选择B（距离2），更新邻接点：C(min(4, 2+1)=3), D(2+7=9)
    S = {A, B}
    
    步骤3：选择C（距离3），更新邻接点：D(min(9, 3+3)=6), E(3+5=8)
    S = {A, B, C}
    
    步骤4：选择D（距离6），更新邻接点：E(min(8, 6+1)=7)
    S = {A, B, C, D}
    
    步骤5：选择E（距离7）
    S = {A, B, C, D, E}
    
    最短距离：
    A到A: 0
    A到B: 2
    A到C: 3
    A到D: 6
    A到E: 7

### 四、证明题
13. 证明：
    设G有n个顶点，ε条边，r个面。
    由欧拉公式：n - ε + r = 2，所以r = ε - n + 2。
    
    由于每条边至多在两个面的边界中出现，每个面的边界包含至少l条边，
    所以所有面的边界边数之和 ≥ r×l。
    同时，这个和 = 2ε（每条边在边界中恰好出现2次，对于平面图）。
    
    所以 2ε ≥ r×l = (ε - n + 2)×l
    2ε ≥ lε - l(n-2)
    2ε - lε ≥ -l(n-2)
    ε(2-l) ≥ -l(n-2)
    ε(l-2) ≤ l(n-2) （注意l≥3，所以l-2>0）
    ε ≤ l(n-2)/(l-2)

14. 证明：
    方法一（组合意义）：
    左边Σk·C(n,k)表示在n个人中选k个人组成委员会，并从中选一个领导的方案数总和。
    这等价于：先从n个人中选一个当领导（n种方法），再从剩余的n-1个人中任选一些作为委员（2^{n-1}种方法）。
    所以Σk·C(n,k) = n·2^{n-1}。
    
    方法二（微分法）：
    由二项式定理：(1+x)ⁿ = ΣC(n,k)x^k
    两边对x求导：n(1+x)^{n-1} = Σk·C(n,k)x^{k-1}
    令x=1：n·2^{n-1} = Σk·C(n,k)
