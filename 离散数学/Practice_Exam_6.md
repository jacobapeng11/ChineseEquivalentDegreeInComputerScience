   - r, r³: 3¹ = 3 (4条边同色)
   - r²: 3² = 9 (对边同色)

   重新计算：
   - e: 3⁴=81
   - r, r³: 3¹=3
   - r²: 3²=9
   - 对角线翻转2个：3²=9 (每种)

   - e: 3⁴=81
   - r, r³: 3¹=3
   - r²: 3²=9
   - 2个对角线翻转: 2×3²=18
   - 2个对边中点连线翻转: 2×3²=18
   
   结果 = (81+3+9+3+18+18)/8 = 132/8 = 16.5，仍不对。
   
   让我重新详细分析对角线翻转：
   顶点1,2,3,4顺时针，则边1-2,2-3,3-4,4-1
   关于1-3对角线翻转：边1-2↔边4-3即3-4，边2-3↔边4-1。

### 三、计算题
11. 解：
    Prim算法从任意顶点开始，逐步扩展最小生成树：
    起始点A，当前树={A}
    1. 选择与A关联的最短边(A,B)=1，树={A,B}
    2. 选择与{A,B}关联的最短边，(B,C)=2或(A,D)=3，选(B,C)=2，树={A,B,C}
    3. 选择与{A,B,C}关联的最短边，(C,D)=1，树={A,B,C,D}
    4. 选择与{A,B,C,D}关联的最短边(D,E)=2，树={A,B,C,D,E}
    
    最小生成树的边集：{(A,B), (B,C), (C,D), (D,E)}，总权重=1+2+1+2=6

12. 解：
    齐次方程：aₙ = 6a_{n-1} - 9a_{n-2}
    特征方程：x² - 6x + 9 = 0，(x-3)² = 0，x = 3（二重根）
    齐次通解：aₙ^h = (α + βn)·3ⁿ
    
    非齐次特解：由于非齐次项是2·3ⁿ，且3是二重特征根，设特解aₙ^p = γn²·3ⁿ
    代入原方程：γn²·3ⁿ = 6γ(n-1)²·3^{n-1} - 9γ(n-2)²·3^{n-2} + 2·3ⁿ
    两边同除以3^{n-2}：9γn² = 18γ(n-1)² - 9γ(n-2)² + 18
    展开：9γn² = 18γ(n²-2n+1) - 9γ(n²-4n+4) + 18
    = 18γn² - 36γn + 18γ - 9γn² + 36γn - 36γ + 18
    = 9γn² - 18γ + 18
    比较系数：左边9γn²，右边9γn²，常数项：0 = -18γ + 18
    所以γ = 1
    
    通解：aₙ = (α + βn)·3ⁿ + n²·3ⁿ = (α + βn + n²)·3ⁿ
    
    初始条件：
    a₀ = α = 1
    a₁ = (α + β + 1)·3 = (1 + β + 1)·3 = 3(2 + β) = 6 + 3β = 5
    所以3β = -1，β = -1/3
    
    所以aₙ = (1 - n/3 + n²)·3ⁿ

### 四、证明题
13. 证明：
    用反证法。假设图G不连通，则G至少包含两个连通分量。
    
    设G的一个连通分量有k个顶点（1≤k≤n-1），则这个连通分量最多有C(k,2) = k(k-1)/2条边。
    剩余的n-k个顶点最多有C(n-k,2) = (n-k)(n-k-1)/2条边。
    
    所以总的边数m ≤ k(k-1)/2 + (n-k)(n-k-1)/2
    
    令f(k) = k(k-1)/2 + (n-k)(n-k-1)/2，其中1≤k≤n-1
    f(k) = (k²-k)/2 + ((n-k)²-(n-k))/2
    = (k²-k+n²-2nk+k²-n+k)/2 = (2k²-2nk+n²-n)/2
    = k² - nk + (n²-n)/2
    
    这是关于k的二次函数，开口向上，对称轴为k=n/2
    所以在k=1或k=n-1处取得最大值
    
    当k=1时，f(1) = 1-n+(n²-n)/2 = 1-n+(n²-n)/2 = (2-2n+n²-n)/2 = (n²-3n+2)/2 = (n-1)(n-2)/2
    当k=n-1时，f(n-1) = (n-1)² - n(n-1) + (n²-n)/2 = (n-1)(n-1-n) + (n²-n)/2 = -(n-1) + (n²-n)/2 = ...
    = (n-1)² - n(n-1) + (n²-n)/2 = n²-2n+1 - n²+n + (n²-n)/2 = -n+1 + (n²-n)/2
    = (n²-n-2n+2)/2 = (n²-3n+2)/2 = (n-1)(n-2)/2
    
    所以m ≤ (n-1)(n-2)/2，这与题设m > (n-1)(n-2)/2矛盾
    
    所以原假设错误，图G是连通的

14. 证明：
    这是著名的Dobinski公式或Stirling数的下降阶乘幂展开。
    
    左边xⁿ表示用x种颜色给n个位置着色的方法数（每个位置可用任意颜色）。
    
    右边ΣS(n,k)·x(x-1)...(x-k+1)按如下方式解释：
    - S(n,k)表示将n个位置划分为k个非空组的方法数
    - x(x-1)...(x-k+1)表示给这k个组着色的方法数（第1组x种选择，第2组x-1种，...，第k组x-k+1种），即k-排列P(x,k)
    
    组合意义：考虑n个位置着色的所有方案，按实际使用了多少种颜色分类。
    如果使用了k种不同的颜色，那么：
    1. 将n个位置按颜色分组，形成k个非空子集，方法数为S(n,k)
    2. 从x种颜色中选k种并排列，方法数为P(x,k) = x(x-1)...(x-k+1)
    
    对所有可能的k值（从0到n）求和，得到所有着色方案数xⁿ。
    
    因此xⁿ = Σ_{k=0}^n S(n,k)·x(x-1)...(x-k+1)
