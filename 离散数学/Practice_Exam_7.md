6. 6 - (p→q)∧(q→r)→(p→r)是重言式，其真值表全为T，所以没有成假赋值，主合取范式为空积（即为T），不含极大项。等等，重言式在主合取范式中对应于无极大项，但按定义应是空合取，即T。实际上，重言式的主合取范式包含所有使其为假的赋值对应的极大项。但重言式无成假赋值，所以主合取范式应为空。但如果考虑所有可能的变量组合（p,q,r共8种），重言式在这8种赋值下都为真，所以没有极大项。不过按定义，主合取范式包含使公式为假的极大项，重言式无成假赋值，故0个极大项。但是，一般约定重言式的主合取范式为1（即空合取），在3个变量情况下，这通常解释为公式等价于所有2³=8个极大项的合取中去掉那些使公式为真的项。让我具体分析：(p→q)∧(q→r)→(p→r)是一个包含3个变量的公式，共有2³=8个可能的真值赋值，这个公式在所有8种赋值下都是真的，所以没有成假赋值，因此主合取范式中包含0个极大项。但按约定，重言式的主合取范式是所有极大项的合取中去掉成真项后的部分，即无。如果按传统说法，重言式的主合取范式是空合取，即真值为T，不包含任何极大项。在3变量情况下，如果一个公式是重言式，那么它的主合取范式不包含任何极大项，所以答案是0。但这是一个陷阱题，因为(p→q)∧(q→r)→(p→r)是重言式，主合取范式为空，所以是0个极大项，但没有这个选项。让我确认这确实是重言式：在所有8种赋值下都为真。所以主合取范式不含极大项。答案应为0，但若选项中有其他解释，则重言式的主合取范式通常被视为不包含任何极大项。答案是0。

   实际上，让我验证：(p→q)∧(q→r)→(p→r)是假的情况只在(p→q)∧(q→r)为真且(p→r)为假时发生。即p真r假，且p→q和q→r都真。如果p真r假，则q必须真才能使p→q真，q必须真才能使q→r真，但q真p真r假则q→r为假。所以不存在使公式为假的赋值。所以公式是重言式，有0个极大项。

### 三、计算题
11. 解：
    由Lagrange定理，|G|=6，|H|=2，则H在G中的指数为|G|/|H|=6/2=3。
    所以G中H的左陪集有3个，设为H, aH, bH（其中a,b∉H且aH≠bH）。
    
    商集G/H = {H, aH, bH}构成群当且仅当H是G的正规子群。
    证明：
    必要性：若G/H构成群，则陪集运算(cH)(dH)=(cd)H是良定义的。
    对任意g∈G，h∈H，有gH·hH = (gh)H和H·gH = (g)H
    由于H是单位元陪集，需要gH·H = gH，这要求对任意h∈H，ghg⁻¹∈H
    所以gHg⁻¹⊆H，又|gHg⁻¹|=|H|，所以gHg⁻¹=H，即H⊴G。
    
    充分性：若H⊴G，则对任意g∈G，有gH=Hg。
    对于陪集乘积(aH)(bH)，定义为(ab)H是良定义的，因为如果aH=a'H，bH=b'H，
    则存在h₁,h₂∈H使得a'=ah₁, b'=bh₂，所以a'b'=(ah₁)(bh₂)=a(h₁b)h₂=ab(b⁻¹h₁b)h₂
    由于H是正规子群，b⁻¹h₁b∈H，所以a'b'H=ab(b⁻¹h₁b)h₂H=abH
    所以运算良定义，G/H构成群。

12. 解：
    设序列{2n+1} = {1, 3, 5, 7, ...}的普通生成函数为G(x) = Σ(2n+1)xⁿ，n从0到∞
    
    G(x) = Σ(2n+1)xⁿ = Σ2nxⁿ + Σxⁿ
         = 2Σnxⁿ + Σxⁿ 
         = 2xΣnx^{n-1} + 1/(1-x)
         = 2x·d/dx[Σxⁿ] + 1/(1-x)
         = 2x·d/dx[1/(1-x)] + 1/(1-x)
         = 2x·1/(1-x)² + 1/(1-x)
         = 2x/(1-x)² + (1-x)/(1-x)²
         = (2x + 1 - x)/(1-x)²
         = (1+x)/(1-x)²

### 四、证明题
13. 证明：
    设G是n个顶点、n-1条边且无回路的无向图。
    
    需要证明G连通。
    
    用反证法：假设G不连通，则G至少有两个连通分量。
    设G的连通分量为G₁, G₂, ..., Gₖ (k≥2)，各含有n₁, n₂, ..., nₖ个顶点，且n₁+n₂+...+nₖ=n。
    
    由于每个连通分量Gᵢ无回路且有nᵢ个顶点，所以Gᵢ是树，有nᵢ-1条边。
    所以G的总边数 = (n₁-1) + (n₂-1) + ... + (nₖ-1) = (n₁+n₂+...+nₖ) - k = n - k
    
    但题设G有n-1条边，所以n-1 = n-k，即k=1
    
    这与k≥2矛盾，所以G必须连通。
    
    所以G是连通无回路图，即G是树。

14. 证明：
    S(n,k)表示将n个不同的元素分成k个非空子集的方法数。
    
    考虑第n个元素的归属：
    1) 第n个元素单独组成一个子集：其余n-1个元素需要分成k-1个非空子集，方法数为S(n-1,k-1)
    2) 第n个元素加入到已有的k个子集之一中：先将n-1个元素分成k个非空子集，方法数为S(n-1,k)，
       然后将第n个元素加入k个子集中的任意一个，方法数为k，总共k·S(n-1,k)种方法
    
    这两种情况包含了全部且互不重复：第n个元素要么单独成组，要么加入已有组。
    
    所以S(n,k) = S(n-1,k-1) + k·S(n-1,k)
