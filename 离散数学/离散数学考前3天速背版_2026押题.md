# 离散数学考前 3 天速背版：2026 押题版

> 用法：这不是教材，是最后 3 天的取分清单。
>
> 目标：不超纲，只背 2026 最可能直接兑现的模板和结论。
>
> 公式集中看：[离散数学考前速背_公式模板.md](离散数学考前速背_公式模板.md)

## 一、先记结论：2026 最可能怎么考

### 1. 最可能结构

- 逻辑形式化：4 分
- 选择题：8 到 10 分
- 填空题：6 到 8 分
- 计算 / 应用题：10 分左右
- 证明题：10 分左右

### 2. 最值得押的七块

1. 谓词逻辑形式化
2. 关系、偏序、函数
3. 图论、树、平面图、着色
4. 排列组合与限制计数
5. 递推 / 母函数 / 指数型母函数
6. 集合或组合恒等式证明
7. 公式模板统一背诵

### 3. 最重要的策略判断

- 不要赌“2025 考过了，2026 就不考”。
- 要赌“2025 缺席但 2021-2024 连续高频的模块会回归”，尤其是递推 / 母函数 / 指数型母函数。
- 宏观模块不会消失，变化的是小题型和数字。
- 公式全部集中到模板文件里，不要在速背版里分散记。

## 二、考前 3 天安排

### 第 1 天：逻辑 + 关系 + 偏序

今天只做三件事：

1. 把逻辑形式化的入口背熟。
2. 把等价关系、偏序关系、单射 / 满射 / 双射的判断做通。
3. 把“没有两个 / 至多一个 / 恰好一个”这类句式和函数性质刷一遍。

今天必须会写的入口：

- 谓词逻辑的“所有 / 存在 / 不是每个 / 没有两个 / 至多一个 / 恰好一个”都要能立刻翻成模板。
- 关系题先判等价和偏序，再看函数性质。
- 这些完整公式统一放在 [离散数学考前速背_公式模板.md](离散数学考前速背_公式模板.md)。

关系题必须会判：

- 等价关系：自反 + 对称 + 传递
- 偏序关系：自反 + 反对称 + 传递

偏序题要会看：

- 极大值、极小值
- 最大值、最小值
- 最长链、最长反链

### 第 2 天：图论 + 组合计数

今天只做四件事：

1. 把树、握手定理、平面图欧拉公式背熟。
2. 把生成树、小图计数、无奇圈二部图、着色数刷熟。
3. 把插空、圆排列、男女相间、分组配对、隔板法做一遍。
4. 把过去做错的图论与组合题重做一遍。

今天必须会用的入口：

- 图论只盯四个关键词：握手定理、树的边数、欧拉公式、平面图边数上界。
- 组合计数只盯四个分型：选、排、分组、整数解。
- 这些具体公式统一看 [离散数学考前速背_公式模板.md](离散数学考前速背_公式模板.md)。

组合计数必须会分型：

- 选：先想组合数
- 排：先想排列数
- 分组：记得除组内和组间重复
- 整数解：看“正整数”还是“非负整数”

### 第 3 天：递推 / 母函数 + 证明模板 + 临场模拟

今天只做三件事：

1. 把递推、普通母函数、指数型母函数的入口刷熟。
2. 把集合恒等式、组合恒等式、关系证明模板背下来。
3. 至少完整做 1 套押题卷，按 45 到 50 分钟限时。

递推题最常用两种处理：

- 一阶线性递推先找固定点，再做平移化简。
- 二阶常系数递推先写特征方程，再看是否有重根。

母函数题要分两类：

- 递推型优先普通母函数。
- 位置计数、偶奇筛选优先指数型母函数。
- 具体公式统一看 [离散数学考前速背_公式模板.md](离散数学考前速背_公式模板.md)。

## 三、最该背的 12 个结论

1. 幂集大小要会一眼反应。
2. 等价关系对应划分。
3. 偏序关系就是自反、反对称、传递。
4. 整除关系通常是偏序。
5. 链就是任意两元素可比较。
6. 反链就是任意两元素不可比较。
7. 握手定理是图论基础入口。
8. 树的边数一定比顶点少 1。
9. 连通平面图先想欧拉公式。
10. 无奇圈图先想二部图与 2 着色。
11. 组合计数看到“选”就想组合数，看到“排”就想排列数。
12. 递推 / 母函数 / 指数型母函数是 2026 最值得回看的高频入口。

## 四、证明题直接套的四个模板

### 1. 集合恒等式

- 先证左边包含右边，再证右边包含左边。
- 最稳方法是元素法。

### 2. 组合恒等式

- 优先想同一件事的两种数法。
- 不要上来硬代公式。

### 3. 关系证明

- 等价关系：证自反、对称、传递。
- 偏序：证自反、反对称、传递。
- 划分：证并为全集、两两不交。

### 4. 图论证明

- 证明有回路：反证到树。
- 证明平面图边数上界：先看每个面至少几条边，再联立欧拉公式。
- 证明树至少有两个叶子：取最长路两端点。

## 五、考场取分路线

### 目标 23 分的最稳路线

1. 逻辑形式化：拿 3 到 4 分
2. 选择题：至少拿 6 分
3. 填空题：至少拿 4 分
4. 计算题：至少拿 5 分
5. 证明题：至少拿 4 分

合计：22 到 23 分以上。

### 正式做题顺序

1. 逻辑题
2. 选择题
3. 填空题里公式直出题
4. 计算题里你最熟的一题
5. 最后再写证明题

### 临场不要死磕的内容

如果只剩最后几天，以下内容只看定义，不投入主刷时间：

- 群、环、域
- Burnside 引理
- Polya 定理
- Skolem 标准形

原因不是它们不在考纲里，而是 2017-2025 真题显性出现率太低，不适合最后几天主攻。

## 六、最后一句话

2026 最可能继续考：逻辑形式化 + 关系 / 偏序 + 图论 + 限制计数 + 递推 / 母函数 / 指数型母函数 + 一道模板化证明题。最后 3 天最值钱的，不是再学新知识，而是把这些入口做到“看到题型就知道往哪套”。
